#bug-reports -- If you find a bug with the theme you are using, please post them in this channel with the following information:

  • What theme are you using?
  • Where can did you find this bug?
  • Does this bug exist when you turn the theme on?
  • Is this bug related or connected to a plugin?
#support -- If you have any question about the theme file, how the theme works, how to edit the theme, etc. Please post your concerns in this channel.
#suggestions -- If you think the theme could be better improved by certain changes, feel free to post your suggestions here! If you see an abandoned theme that you think has potential, you can also suggest for the theme devs to adopt it.
#roles -- You can add a role corresponding to the theme you're using from this channel. This helps the theme devs identify what theme you are talking about should you report a bug with insufficient information or no screenshot.
